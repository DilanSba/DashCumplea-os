# Celebration Windmar Hub 🎂

Un centro de celebraciones premium diseñado para gestionar y celebrar los cumpleaños de los talentos de Windmar Home. Con un diseño limpio, moderno y responsivo.

## ✨ Características

- **Billboard de Hoy**: Una tarjeta imponente que celebra al talento del día con fotos de gran tamaño.
- **Dashboard de Próximos Cumpleaños**: Visualiza rápidamente quiénes son los siguientes en celebrar.
- **Calendario Anual**: Vista completa de todos los talentos agrupados por mes.
- **Búsqueda y Filtros**: Encuentra talentos por nombre, área o cargo al instante.
- **Modo Oscuro/Claro**: Adaptado para cualquier preferencia visual.
- **Personalización de Temas**: Cambia el color de acento de toda la plataforma dinámicamente.
- **Recordatorios Visuales**: Indicadores claros para los cumpleaños de hoy y mañana.

## 🚀 Tecnologías Utilizadas

- **React 19**
- **Vite**
- **Tailwind CSS 4**
- **Motion (Framer Motion)** para animaciones fluidas.
- **Lucide React** para iconografía.
- **Google Gemini API** (Opcional, para funciones inteligentes).

## 🛠️ Instalación y Desarrollo

1. **Clonar el repositorio**:
   ```bash
   git clone <URL_DEL_REPOSITORIO>
   cd celebration-windmar-hub
   ```

2. **Instalar dependencias**:
   ```bash
   npm install
   ```

3. **Variables de Entorno**:
   Copia el archivo `.env.example` a `.env` y añade tu `GEMINI_API_KEY` si deseas usar funciones de IA.

4. **Iniciar en modo desarrollo**:
   ```bash
   npm run dev
   ```

5. **Construir para producción**:
   ```bash
   npm run build
   ```

## 🌐 Despliegue en Vercel

Esta aplicación está lista para ser desplegada en **Vercel**. Simplemente conecta tu repositorio de GitHub a Vercel y se detectará automáticamente como un proyecto de Vite.

Asegúrate de configurar las variables de entorno en el panel de Vercel si es necesario.
